Soundnode App is a Open Source project to support Soundcloud for Desktop platform.

You can report bugs on https://github.com/Soundnode/soundnode-app/issues or send me a message on twitter @weblancaster.

Features:
  • - No need to install
  • - Native media keyboard shortcuts
  • - Search for new songs
  • - Easy navigation
  • - Listen to songs from your Stream, Likes, Tracks, Following or Playlists
  • - Like songs and save to your liked playlist
  • - Full playlist feature
  • - Follow/Unfollow users
