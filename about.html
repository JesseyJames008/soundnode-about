Soundnode App is a Open Source project to support Soundcloud for Desktop platform.

This is a beta version therefore the application contain some minor bugs which you can report on https://github.com/Soundnode/soundnode-app/issues

Features included:
  • - Search
  • - Keyboard shortcuts (not native for this release) Play/Pause command+return or space, Prev command+left, Next command+right, Volume down command+down, Volume up command+up, Cheat-sheet shift+? (ps: use ctrl for Windows/PC user)
  • - Listen to songs from your Streams, Likes, Tracks or Playlists
  • - Automatic notification update
